Priority  - is set by parser and type is returned by parser.
Type - is what the parser reports to "analyzer" if something was found.
Sub-type - is passed on by info parameter or desided by "codec" when transform fails or not.

There are 5 levels of priority.

0 - root parser (any below, only one can be set to 0)
1 - known container format (cd, mdf, ...)
2 - recursive types (tiff, gif, tar, ...)
3 - regular type (images, zlib ...)
4 - same as abowe with lower priority
5 - hard to parse type (text, ...)
6 - default (unknown type)

If some parser sets its priority to 0 then all other parsers are disabled, exept default parser (priority 6).
