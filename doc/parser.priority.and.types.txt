Priority  - is set by parser and type is returned by parser.
Type - is what the parser reports to "analyzer" if something was found.
Sup-type - is passed on by info parameter or desided by "codec" when transform fails or not.

There are 5 levels of priority.

0 - known container format (tar, cd, mdf, ...)
1 - recursive types (zlib, bzip, ...)
2 - regular type (images, ...)
3 - same as abowe with lower priority
4 - default (unknown type) or hard to parse type (text, ...)
